
# Employees--segmentation--Analysis
Executed funnel analysis in Power BI to monitor employees segmentation and workplace satisfication
Query; The difference between workplace satisfication for different departments of the company.
Tools used; a) cleaned & transformed the data- Power Query tools, b) meta data analysis- DAX formulas/ DAX queries, c) representation- visualisation tools; [button slicers, Line series, KPI's, Pie-Chart, Stacked-Bar-Chart & others ]
Insights- *measure of workplace satisfication in different departments
*Gender wise divison of employees & workplace satisfication
*Employees segmentation in different job role for different departments.
